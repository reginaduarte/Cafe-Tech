
# Café Tech – Criando um agente com inteligência artificial (QA)

Este repositório foi criado como parte do **Café Tech – HUB QA**, apresentado por  
**Regina Duarte (Quality Engineering)**, com o tema:

> **Criando um agente com inteligência artificial aplicado ao contexto de Qualidade de Software**

Neste momento, o repositório contém **apenas testes automatizados com Playwright**, que serão utilizados como base prática para a criação e evolução de um **Agente de IA para QA**.

---

## 🎯 Objetivo do repositório

- Disponibilizar um exemplo simples de **testes automatizados com Playwright**
- Servir como base para:
  - Code review de testes
  - Identificação de flakiness
  - Discussões sobre qualidade e boas práticas
- Evoluir futuramente para incluir:
  - Um **Agente QA Reviewer**
  - Prompts reutilizáveis
  - Automação de revisões de Pull Request com apoio de IA

---

## 🧪 Tecnologias utilizadas

- **Playwright**
- **TypeScript**
- **Node.js**
